# #1: How to Inspect and Select an Element on a Web Page

This guide explains how to select and inspect any element on a webpage using **Browser (Inspect) / Developer Tools** (Chrome, Edge, Brave, Firefox).

---

## Step 1: Open the Web Page
Open the website where you want to inspect an element.

---

## Step 2: Open Developer Tools
You can open DevTools using **any one** of the following methods:

- Right-click anywhere on the page → click **Inspect**
- Keyboard shortcuts:
  - **Windows / Linux:** `Ctrl + Shift + I`
  - **Mac:** `Cmd + Option + I`

Developer Tools will open on the right or bottom of the screen.

---

## Step 3: Enable Element Selection Tool
In DevTools, click the **Select Element** icon  
(it looks like an arrow inside a box, located at the top-left).

### Shortcut:
- **Windows / Linux:** `Ctrl + Shift + C`
- **Mac:** `Cmd + Shift + C`

---

## Step 4: Hover Over the Page
Move your mouse over the webpage.

- Elements will highlight as you hover
- You’ll see spacing details like margin, padding, and size

---

## Step 5: Click the Element
Click on the element you want to inspect (text, image, button, div, etc.).

- The corresponding **HTML code** will be automatically selected in the **Elements** tab.

---

## Step 6 (Optional): Explore or Edit
After selecting an element, you can:

- Edit HTML (double-click text)
- Modify CSS in the **Styles** panel
- Add or remove classes
- Copy selector or XPath

---

## Important Notes
- Changes made in DevTools are **temporary**
- Refreshing the page will reset everything
- Hovering over HTML in DevTools highlights the element on the page

---

## Helpful Tips
- Right-click on a specific element → **Inspect** to jump directly to it
- Use DevTools to understand layout, spacing, and styling
- Great for learning HTML, CSS, and debugging UI issues
